# H3 Гексагональная сетка - Быстрый старт

## Что реализовано

Добавлена возможность отображения **гексагональной сетки H3** на карте в разделе ServiceArea.

## Как использовать

### На основной карте ServiceArea

1. Откройте **ServiceArea** в Sonata Admin
2. Перейдите на вкладку **"Geo Areas"**
3. Над картой включите **переключатель "Показать гексагональную сетку"**
4. Настройте **детализацию** с помощью слайдера (5-10)

### В модальном окне создания кастомной зоны (интерактивный выбор)

1. Нажмите кнопку **"Создать кастомную зону"**
2. В открывшемся модальном окне включите **переключатель "Показать гексагональную сетку"**
3. Настройте **детализацию** с помощью слайдера
4. **Кликайте на гексагоны** чтобы выбрать их (зеленый цвет = выбран)
5. Кликните повторно на выбранный гексагон чтобы отменить выбор
6. Введите название и сохраните
7. **Выбранные гексагоны сохранятся как MultiPolygon** (набор отдельных гексагонов)

**ИЛИ** используйте стандартное рисование полигона инструментами Leaflet.draw

## Управление

- **Переключатель** - включение/выключение сетки
- **Слайдер** - изменение размера гексагонов (5 = крупные, 10 = мелкие)
- **Информация** - показывает текущий размер гексагона

## Особенности

✅ Автоматическое обновление при перемещении карты
✅ Динамическое изменение разрешения
✅ **Интерактивный выбор гексагонов кликами** (в модальном окне)
✅ Визуальная обратная связь (зеленый = выбран, оранжевый = доступен)
✅ Счетчик выбранных гексагонов
✅ Кнопка "Очистить" для сброса выбора
✅ Автоматическое создание MultiPolygon из выбранных гексагонов
✅ Оптимизация производительности (макс. 2000 гексагонов)
✅ Hover эффекты на гексагонах
✅ Следование принципам SOLID и идиомам Symfony/Sonata/Stimulus

## Рекомендуемые разрешения

| Разрешение | Размер | Применение |
|------------|--------|------------|
| 5-6 | Крупные гексагоны | Для больших регионов |
| 7-8 | Средние гексагоны | Для городов и районов |
| 9-10 | Мелкие гексагоны | Для детальной работы |

## Архитектура

Решение построено на 3 основных компонентах:

1. **HexagonService** - работа с H3 библиотекой
2. **MapService** - управление гексагональными слоями на карте
3. **geo_area_map_controller** - Stimulus контроллер для UI

Все компоненты следуют принципам **SOLID** и могут быть расширены без изменения существующего кода.
